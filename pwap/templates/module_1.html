Borders

Borders determine what appears along the outside edges of an element. Using CSS rules, we can style an element's border in many different ways.

A border has three basic components:

  • Width
  • Style (e. g. solid, dashed)
  • Color

An element's border is one of its basic box model components, which means it adds to the element's size. When calculating the overall size of an element, the width of its border must be taken into account.

Example
I have a border!

border: 1px solid blue;
