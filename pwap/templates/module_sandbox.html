Margins

                            

                            

                                

                                    

                                        
We've now learned all the attributes necessary to set the overall size of an element as well as its basic appearance. When we put all these attributes together, we have what's called the Box Model.

                                        
Conceptually, the Box Model consists of four parts (courtesy of w3schools):

                                        
    
                                            
  • Content: the content of the box, where text and images appear.
    • 
                                            
  • Padding: clears an area around the content. The padding is transparent.
    • 
                                            
  • Border: a border that goes around the padding and content.
    • 
                                            
  • Margin: clears an area outside the border. The margin is transparent.
    • 
                                        

                                        
This translates to five CSS attributes that work together to make an element's Box Model:

                                        
    
                                            
  • The width and height attributes set the size of the content.
    • 
                                            
  • The padding attribute sets the space between the content and the outer edge of the element.
    • 
                                            
  • The border attribute sets the size, style, and color of the border surrounding the outer edge of the element.
    • 
                                            
  • The margin attribute sets the space between the outer edge of the element and surrounding elements.
    • 
                                        

                                        
The total amount of space the element takes up on the page is the sum of its width and height, padding, margin, and border. For example, to find the total horizontal "footprint" of an element, add its width, right and left padding, right and left border width, and right and left margin.

                                    

                                    

                                        
Example

                                        
The styling for this paragraph uses all five box model attributes. Notice that the total width of the element is 200 + 2*1 + 2*20 + 2*20 = 282 pixels.

                                        

                                        
width: 200px;
height: 200px;
border: 1px solid blue;
padding: 20px;
margin: 20px;
