To start writing the CSS, we need a selector that says what kind of element we want to style. Here our only element is a div, so we'll select the div by writing div { }.

Now we need to start writing rules inside our selector. First, let's look at font. The font in the goal image is Courier. To make our font courier, add this rule: font-family: Courier;

Finally, let's add the colors. In the goal image, the text is white and the background is red. To make the text white, add this rule: color: white;. To make the background red, add this rule: background-color: red;

Borders

Borders determine what appears along the outside edges of an element. Using CSS rules, we can style an element's border in many different ways.

A border has three basic components:

  • Width
  • Style (e. g. solid, dashed)
  • Color

An element's border is one of its basic box model components, which means it adds to the element's size. When calculating the overall size of an element, the width of its border must be taken into account.

Example
I have a border!

border: 1px solid blue;
